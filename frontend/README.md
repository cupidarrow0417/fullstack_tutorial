# Flask Back-end Deployment

- Author: Jack Jiang
- Date: Jul. 2019

## Setup

```shell
npm install
```

## Run

```shell 
npm start
```

## Install new-package

```shell
npm install --save new-package
```

## Update [GitHub Pages](https://jiangyiqun.github.io/fullstack_tutorial/)

```shell
npm run deploy
```
